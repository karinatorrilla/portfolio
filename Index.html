Proyectos

Sistema de gestión de Artículos

Desarrollé una aplicación de escritorio para la gestión integral de artículos, categorías y marcas para un catálogo comercial. La solución permite listar, buscar, agregar, modificar, eliminar y detallar artículos, incluyendo múltiples imágenes y persistiendo la información en una base de datos existente.

Panel de gestión de Real State

Plataforma diseñada para la gestión integral de brokers y sus propiedades, permitiendo administrar la información de las propiedades, asignarlas a brokers y publicarlas. El sistema optimiza el flujo de trabajo de publicación y administración inmobiliaria.
